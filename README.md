# OOP-345

C++ Standards
- C++98: [ISO-IEC 14882-1998](https://www.iso.org/standard/25845.html)
- C++03: [ISO-IEC 14882-2003](https://www.iso.org/standard/38110.html). You can read for free a working draft [N1638](https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2004/n1638.pdf)
- C++11: [ISO-IEC 14882-2011](https://www.iso.org/standard/50372.html). You can read for free the final draft [N3242](https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2011/n3242.pdf).
- C++14: [ISO-IEC 14882-2014](https://www.iso.org/standard/64029.html). You can read for free the final draft [N3797](https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2013/n3797.pdf).
- C++17: [ISO-IEC 14882-2017](https://www.iso.org/standard/68564.html). You can read for free the final draft [N4659](https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2017/n4659.pdf).
- C++20: [ISO-IEC 14882 2020](https://www.iso.org/standard/79358.html). You can read for free the final draft [N4849](https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2020/n4849.pdf).
- C++23: [ISO-IEC 14882 2024](https://www.iso.org/standard/83626.html). You can read for free the final draft [N4950](https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2023/n4950.pdf).
- C++26: Under Review. You can read for free the final draft [N5046](https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2026/n5046.pdf).
